Crear un Usuario

Para una nuevo usuario se deberá hacer clic en el botón Crear ubicado en el ángulo superior izquierdo de la pantalla (vista Lista).

Una vez que se haya hecho clic en el botón mencionado, accederemos a la siguiente pantalla en la que deberán completarse los siguiente datos:

•Nombre: nombre de la persona que se registrará como usuario.

•Usuario: nombre que utilizará el sistema para identificar al usuario.

•Compañía: se seleccionará la compañía para cual trabaja el usuario.

•Activo: se seleccionará esta casilla cuando se desee visualizar el usuario.

Haciendo clic en la imagen se podrá importar una foto del usuario.
