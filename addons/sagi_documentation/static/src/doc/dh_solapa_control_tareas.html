Control de Tareas

Independientemente de la vista, se mostrara en cada proyecto un semáforo, Verde, Amarillo o Rojo según el estado del mismo.

A continuación se describen los colores y las advertencias posibles:

•Overall status: Es el mostrado en la vista kanban o lista del proyecto

oRed, Critical: Al menos una alarma de esta tarea esta en estado Critical state,

oYellow, Warning: Al menos una alarma de esta tarea esta en estado Warning state

oGreen, Ok,

En la vista formulario, en la solapa Control se describe el detalle del estado.

•Execution Status

oRed, Critical Delay: La tarea no esta terminado y la fecha actual es posterior que la fecha "Late Finish Date"

oYellow, Delay: La tarea no esta terminado y la fecha actual es posterior que la fecha "Date to End"

oGreen, On Time,

•Remaining work:

oRed, Critical: Las horas restantes para esta tarea son menores que las disponibles para el recurso asignado desde el momento actual y hasta la "Late Finish Date",

oYellow, Warning:  Las horas restantes para esta tarea son menores que las disponibles para el recurso asignado desde el momento actual y hasta la "Date to End",

oGreen, Ok

Planning

oRed, Error: El entregable no puede ser planificado con las actuales restricciones.

oGreen, Ok

Restrictions could be predecessors, successors, earliest start date and/or latest finish date
