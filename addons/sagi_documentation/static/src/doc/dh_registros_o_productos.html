Registros o Productos

Cabe distinguir que un producto y un registro son cosas diferentes en el sistema:

Productos

Se refiere a un producto o servicio que puede comprarse, moverse, venderse, etc. Puede encontrar más información de productos en Productos.

Los productos presentan los tipos: componente, técnico o formulado.

Registros

Los registros en el sistema se utilizan para identificar procesos de registro de un producto (o un laboratorio) en alguna entidad y bajo algunas condiciones.

Un registro puede involucrar a uno o más producto según sea el caso. Además un producto puede tener más de un proceso de registro, por ejemplo, el mismo producto podría tener los siguientes registros asociados:

•Registro de producto formulado - Alta

•Registro de producto formulado - Agrotec

•Registros temporario (RET)
