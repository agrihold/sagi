Crear registros

Para crear un nuevo registro hacer clic en "Create"

 

Automaticamente se agrega un renglón en modo edición.

Desde aqui,

•Seleccionar el producto del listado. Se despliega una lista con los productos coincidentes con el texto ingresado en el campo.

•Comercial name. Esta información se completa automaticamente si el producto tiene asignado un Nombre Comercial para la Operative seleccionada. Para asignar o modificar un Commercial Name ver Comercial name

•Price Date: Por defecto se autocompleta con la fecha actual, pero puede modificarse a gusto.

•Precio (USD)

•Price Type: Es el tipo de precio. Se despliega una lista con los Tipos de Precios coincidentes con el texto ingresado en el campo. Ver detalle de donde se crean los Price Types con este link.

•Operative: Es la compañía a la que pertenece el precio del producto que se esta ingresando.

•Term: Se despliega una lista con los Términos de pago coincidentes con el texto ingresado en el campo.

•Supplier: Se puede seleccionar un proveedor.

•Invoice Number: Se puede dejar registro del numero de la factura del proveedor al comprar este producto.

 

NOTA: Si previo a la carga del precio del producto, se encuentra activo un filtro por Price Type, al crear un nuevo registro, se autocompletara el campo Price Type con esta información.
