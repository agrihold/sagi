Users

Grupos de usuarios en "Competitiveness Management"

Para esta sección, existen 3 grupos de usuarios. El tipo de usuario define los permisos que el usuario tiene y que tipo de operaciones puede realizar.

•View:

oPuede visualizar todos los precios registrados de la Operativa desde la que este logueado (de la cual debe tener permiso de uso).

oNo puede hacer ABM de ningún precios. Es decir, no puede ni crear, ni modificar, ni eliminar precios registrados.

oPuede emprimir el Reporte gráfico "Evolucion de precios".

oPuede imprimir el Reporte "Price Analysis".

oNo tiene acceso al menu de configuración.

•User:

oPuede visualizar todos los precios registrados de la Operativa desde la que este logueado  (de la cual debe tener permiso de uso).

oPuede hacer ABM de precios  pertenecientes a todos los Price Type a los que este asignado y de la operativa desde la que esta logueado.

oPuede imprimir el Reporte gráfico "Evolucion de precios".

oPuede imprimir el Reporte "Price Analysis".

oNo tiene acceso al menu de configuración.

•Manager:

oPuede visualizar todos los precios registrados de la Operativa desde la que este logueado  (de la cual debe tener permiso de uso).

oPuede hacer ABM de precios  pertenecientes a todos los Price Type a los que este asignado y de la operativa desde la que esta logueado.

oPuede imprimir el Reporte gráfico "Evolucion de precios".

oPuede imprimir el Reporte "Price Analysis".

oTiene acceso al menu de configuración.

Multi Compañías

Vale aclarar que un usuario podrá acceder o registrar información de aquellas operativas (compañías permitidas) que tenga habilitadas en su configuración, pero solo podra visualizar en pantalla informacion asociada a la Operativa desde la que este logueado.

Usuarios que tienen permiso para la empresa "Agrihold" podrán trabajar con cualquiera de las operativas.

Puede consultar más información de uso de multi-compañías en Multi Compañía

Configurar permiso de usuario

Con el usuario administrador,

 Configuracion/Usuarios/Usuarios, en el campo marcado, seleccionar el tipo de usuario.
